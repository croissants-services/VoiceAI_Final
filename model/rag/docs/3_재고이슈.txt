# 입고하는 방법
입고가 중요한 이유는 발주 누락으로 품절이 발생할 수 있고, 과발주로 인한 복잡성 증가하며, 원가를 추적하기 어렵고, 오배송 또는 불량품이 배송되었을때 해결할 수 없기 때문이다. 
입고하는 방법은 먼저 입고 전용 테블릿으로 입고 페이지를 접속합니다. 왼쪽 사이드바에서 재고 관리를 누르고 안에서 입고를 누릅니다. 다음 화면에서 오른쪽 상단에 창고 별로 보기를 누른 다음, 입고 장에서 원물의 수량을 파악해 입고창이 발주 품목과 비교합니다. 입고 품목을 확인 시 배송 예정일을 확인하며 수량 체크합니다. 다음으로 배송이 온 물품을 클릭하여 입고하고 수량을 체크 한 후 배송 중 수량과 실제 배송 완료된 수량을 비교한 뒤 전체 수량이 같다면 “전체입고”를 누릅니다. 전체 입고 탭이 없는 경우, 2개 이상이 배송 진행 중이므로 해당 품목을 확인하여 해당 제품을 입고합니다. 항상 그날 그날 입고 처리를 하지 않으면 추후 입고 제품이 안 보이는 현상이 발생하기 때문에 주의 부탁드립니다. 마지막으로, 수량과 날짜를 확인하여 전체 입고 처리 또는 상세 입고를 누릅니다. 상세 입고를 하는 이유는 한묶음의 배송이라도 수량이 많아지는 경우 따로 배송될 수 있기 때문입니다. 이럴때엔 상세 입고로 정확한 수량을 입고 처리합니다. 

# 재고 실사 하는법
먼저 재고 실사 페이지 접속합니다. 왼쪽 사이드바에서 재고 실사를 누릅니다. 다음으로
오른쪽 상단에서 원자재나 프랩을 누른 후 왼쪽 상단에서 실사 종류를 지정합니다. 다음으로 새로운 재고 실사를 클릭합니다. 혹시 중간 단계에서 멈췄다가 다시하는 경우, 재고 실사 이어하기를 클릭합니다. 다음으로 실제 재고를 확인해야합니다. 박스가 오픈되면 그 제품은 재고가 없는것입니다. 예를 들어, 총 3박스가 있는데 1박스를 뜯어서 사용중이면 재고는 총 2박스입니다. 현재 재고량이 전과 동일한 경우, 과거와 같음으로 뜹니다. 재고 변화는 있거나 없지만 정확한 갯수를 파악 하였을때, “정확히 셈”으로 뜹니다. 재고 수량이 안전재고 수량보다 월등히 많은 경우, “안셈”으로 뜹니다. 이때 일요일 재고 실사시는 활성화 안됩니다. 실사가 완료 된 경우, 재고 이름이 회색으로 뜨고 실사를 진행 해야하는 경우에는 재고 이름이 검정색입니다. 

# SKOS에서 품절을 걸고 싶어요
영업중 품절을 거는 방법에 대해서 설명 해드리겠습니다.
첫째, 주문관리 페이지의 하단에서 자재 품절을 클릭합니다.
둘째, 품절과 관계된 제품을 드래그로 찾거나 검색으로 찾아서 다음을 클릭합니다.
셋째, 특정 제품을 선택합니다. 이때 무엇 떄문에 품절이 발생하였는지 확인이 가능합니다. 예를 들어, 깐쇼새우_31/40이 품절이라 품절을 걸게되면 깐쇼새우 관련 제품 모두가 품절이 걸리게 됩니다. 

# 입고량과 실제 입고량이 차이가 있습니다
실제 입고량과 물품으로 들어온 입고량이 차이가 있습니다. 각 벤더사마다 한번에 다 보내줄수도 있지만 나누어서 보내어 주는 경우도 있어 혼돈이 될수도 있습니다. 오전 입고량과 오후 입고량으로 따로 구분되어 입고 되는 경우가 있으니 상세 입고를 이용하여 입고 부탁드립니다. 입고 화면에서 전체입고 버튼이 아닌 바탕을 클릭하여 상세 입고 페이지로 들어가 줍니다. 다음으로 “상세 입고”를 누른 후 상세 입고 페이지로 들어가게 되면 몇개가 도착 하였는지 직접 입력이 가능합니다. SPQ 단위를 꼭 확인 하셔서 입력 부탁드립니다. 
입고대상이 아닌 자재가 발송되었습니다.
입고대상이 아닌 자재가 발송된 경우 2가지에 의하여 발생됩니다. 리드타임이 간혹 길게 잡혀서 배송되는 경우가 있습니다. 이에 크루분들이 “배송이 도착했는데 입고가 안되었다”라고 판단하고 모두 입고 처리를 하는경우 뒤늦게 오는 배송제품은 입고 상태가 존재하지 않는다고 판단됩니다. 그렇기 때문에 가능한 그날에 들어온 입고 물품만 체크하여 입고처리를 해주시면 감사하겠습니다. 
현장에서 자재가 없어서 품절이 예상됩니다. 
현장에서 자재 부족으로 품절이 예상되는 경우 2가지 방법이 있습니다. 첫째, 현장에서 바로 필요한 긴급 구매가 필요할때 쓰는 방법입니다. 현장에서 긴급으로 사서 해결 할 수 있는 경우는 근처 마트에서 공수하여 해결이 가능합니다. 만약 참기름, 쌀 등 공산품 위주가 필요할 경우, “재고 실사 대상 물품 구매”로 접속 하여 다음과 같이 입력하면 됩니다. 구매처는 식자재 마트로 선택하고, 배달료가 없으면 해당 항목은 비워 둡니다. 자재 항목에는 ‘참기름’을 입력합니다. 구매 단위는 최소 단위 (1g.1ml)로 입력하며, 예를 들어 10kg짜리 쌀을 구매한 경우 10,000으로 입력합니다. 갯수에는 총 구매 수량을 입력하고, 전체 금액은 18,000으로 작성합니다. 이와 같은 방식으로 입력하면 됩니다. 
현장에서 식자재가 아닌 물품을 구매하는 경우, 다음과 같이 처리합니다.
현장에서 사용하는 물품 중 식자재가 아닌 경우 (예로 드라이버, 다에소에서 구매하는 물품, 음식물 쓰레기 봉투, 쓰레기 봉투 등)은 “기타 지출” 항목을 선택합니다. 구매 후 법인카드로 결제하고, 지출 내역에 물품과 사용처를 간략하게 입력합니다. 총 지출 금액은 구매 합계액을 입력합니다. 
두번째 방법은 현장에서 구매가 가능한 경우 현장에서 구매처리로 품절을 막을 수 있고, 현장에서 구매하기 힘든 물품은 본사에 요청 페이지로 특정 제품을 기입하여 올려주시면 처리 도와드리겠습니다. 

# 자재를 폐기 해야 할 것 같아요.
현장에서 짧은 유통기한 등으로 폐기해야 하는 자재가 발생할 경우 지점_본사에_요청으로 남겨주시면 감사하겠습니다. 금일 기준으로 날짜가 지난 자재를 발견하시면, “요구르트 48개 1월14일자 폐기하겠습니다”의 메세지와 함께 폐기할 요구르트 48개를 사진으로 찍어서 함께 첨부해주시면 감사하겠습니다. 이와 같은 이유는 폐기량을 추적하여 안전재고를 적절하게 수정할 수 있기 때문입니다. 그리고, “간혹 오뎅 유통기한이 짧아 소비하는것보다 폐기가 더 많습니다. 혹시 냉장으로만 사용하는 이유가 있을까요? 사각어묵 냉동으로 판매하는데 유통기한도 더 깁니다. 해동도 수제비처럼 소량으로 식스바트에 소분해서 냉장해동하면 괜찮을 것 같은데 좋은 의견 부탁드립니다”와 같이 좋은 의견이 있다면 남겨주시는 것도 감사합니다. 

# 다른 자재가 왔어요.
자재가 잘못 도착한 경우 처리 방법. 예를 들어, 400g짜리 토마토 캔이 도착해야 하는데 4kg짜리 캔이 도착하거나, 타 지점에 도착해야 할 배너나 기물이 잘못 도착한 경우가 있을때, “토마토 입고 처리는 안 했는데 24개짜리로 와야 하는데 4kg짜리로 6개가 왔는데 어떻게 할까요?”와 같은 메세지와 잘못 도착한 물품의 사진과 함께 본사 요청방에 올려주시면 처리 도와드리겠습니다. 
본사에서 “죄송합니다! 사이즈 착오가 있었습니다. 반품 처리하고 재주문 해드리겠습니다. 박건우님, 잘못 온 바스켓 박스를 한쪽에 놔두시면 감사하겠습니다. 반품 요청 후 다시 말씀드릴게요! 박스에는 ‘롯데 택배 네이버 반품’이라고 적어서 문 밖에 놔주시면 감사하겠습니다.”와 같은 메세지를 보내면, “네, 확인 및 조치하였습니다”와 같은 메세지와 함께 잘못 도착한 물품을 포장하여 반품 준비한 사진을 첨부해주시면 감사하겠습니다.
