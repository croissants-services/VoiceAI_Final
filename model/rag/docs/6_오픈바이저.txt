# 새로 산 태블릿PC 세팅하기

## 아래 내용은 샤오미 레드미 프로로 작성된 내용입니다.
샤오미 관련된 것들은 샤오미가 아닌 제품에는 해당하지 않습니다.
안드로이드 패드면 큰 틀에서는 비슷하지만, 세부적으로는 다를 수 있습니다.

1. 먼저 태블릿과 충전기가 상자안에 잘 들어있는지 확인
2. 전원을 켜고, 부팅할 때 설정하는거 한국어, 대한민국으로 세팅 (전면 카메라를 위로 가게 한 후에, 왼쪽 면 위쪽에 있는 작은 버튼을 꾹 누르면 켜짐)
3. 부팅할때 묻는 설정
- 네트워크 연결 → 건너뛰기
- 화면 잠금 → 건너 뛰기
- 사용자 설정하는 페이지 → 전부 끄고 확인
- 그 외 다른 설정들: 건너 뛸 수 있으면 건너뛰기
4. 부팅이 완료되면 디스플레이 설정을 세팅합니다.
5. SKOS 설치를 위해 먼저 USB 디버깅 모드를 켭니다.
- USB 디버깅 모드 켜는 법
    - 샤오미 코리아에서 아이디를 하나 만듭니다. 이미 만든게 있으면 그걸 사용해도 됩니다.
    - 와이파이를 연결
    - 설정 → 태블릿 정보 → OS 버전을 여러번 클릭하여 개발자 모드 활성화
    - 설정 → Xiaomi 계정 → 샤오미 코리아에서 만든 아이디로 로그인
    - 개발자 모드 활성화 되었으면, 설정 → 일반 → 개발자 옵션 → “USB 디버깅” 켜기
    - 이 때 10초정도 기다렸다가 결과 받아들인다는거 클릭 후 확인
    - “USB 디버깅” 바로 3번째 아래에 있는 “USB로 설치” 켜기
        - 이 때, 에러가 뜨면서 USB 설치가 안켜질 수 있습니다.
        - 이럴 때는, 샤오미 코리아에서 만들어뒀던 아이디를 삭제하고, 다시 만들어서 로그인하시면 됩니다.
        - 참고로 같은 이메일로 아이디를 삭제하고 만드는걸 반복할 수 있습니다.
    - 설정 → 검색에서 알 수 없는 앱 설치 → 파일탐색기 찾아서 허용으로 변경
    - 설정 → Xiaomi 계정 → 로그아웃
6. SKOS 설치 파일을 담은 USB를 준비합니다.
- 해당 USB는 오픈바이저가 가지고 있습니다.
7. USB를 태블릿PC에 연결하고 파일탐색기를 통해서 USB로 들어갑니다.
- 이 때 뜨는 모든 권한은 다 허용합니다.
8. app-debug.apk 파일을 실행합니다.
9. 설치를 완료하고 나면 바탕화면에 SKOS가 생성되었음을 확인
10. SKOS에 들어가면, 위치 관련 권한 설정 허용
11. bd / 1234 로 로그인
12. 오른쪽 위에 지점을 사용하려는 지점으로 변경
13. 함께 들어있던 충전기를 연결하여 태블릿을 알맞은 위치에 설치하고, 남은 상자는 버려도 됨.


# 새 컴퓨터 윈도우 설치하기

주의!! 윈도우 설치해서 바탕화면 보이기 전까지는 "절대" 인터넷 연결하면 안됩니다.
1. 컴퓨터에 모니터, 키보드, 마우스 연결합니다.
- 이 때, 모니터는 컴퓨터의 그래픽카드에 꼽습니다.
- 메인보드 쪽 포트(USB, LAN 근처 포트)에는 연결하면 안됩니다.
2. 모니터의 전원을 켭니다. (컴퓨터는 켜지 않습니다.)
3. 오픈바이저 USB를 컴퓨터에 꼽습니다.
4. 컴퓨터를 켭니다.
- 조금 기다리면 윈도우 설치창이 뜹니다.
5. 윈도우 설치창에서 다음을 누르다보면, 제품키를 입력하라고 뜨는데, 제품키 없음을 클릭합니다.
6. 윈도우 선택에서는 Windows 10 Home을 선택합니다.
7. 업그레이드 vs 사용자 지정에서 사용자 지정을 선택합니다.
8. 설치가 진행되다가 재부팅이 되는데, 기다리면 설정창으로 넘어갑니다.
9. 이 때, 한국 → 키보드 관련된건 기본으로 설정되어있는애들 이렇게 선택해주시면 됩니다.
10. 네트워크 연결하라고 뜨는데, 네트워크 없이 진행하는 버튼을 누릅니다.
11. 그 후에 제한된 설치를 누릅니다.
12. PC 사용자는 ‘블리츠다이나믹스’로 입력해주세요
13. 비밀번호와 보안질문은 적당한걸 입력해주고, 비밀번호는 지점 사람들에게 공유해주세요.
14. 그 후 권한같은걸 묻는데, 전부 아니오를 누르고 진행해주세요.
15. 조금 기다리면 설치가 완료됩니다. 이 때 USB를 제거해주시면됩니다.


# 새로 산 영수증 출력기 설치하기

준비물: EPSON 영수증 출력기, 주방 공유기로 인터넷에 연결된 컴퓨터

영수증 출력기를 받으면, 별도 상자에 이것저것 들어있는데 전원공급장치와 테스트용 용지만 빼고 나머지는 버립니다.

1. 공유기에와 영수증 출력기를 랜선으로 연결합니다. (모뎀에 꽂으면 안됩니다. 공유기여야합니다.)
2. 영수증 출력기의 뚜껑을 엽니다.
3. 테스트용 용지를 출력기에 넣어서 테스트 영수증이 출력될 수 있도록 합니다.
4. 영수증 출력기의 뚜껑을 닫습니다.
5. 전원공급장치를 영수증 출력기에 연결하고 전원을 켭니다.
- 전원공급장치에 연결하는 파워선은 확장팀이 별도로 준비했으니, 못찾으시면 확장팀에게 문의해주세요.
6. 공유기와 연결이 잘 되었다면, 조금 기다렸을 때 영수증이 하나 출력됩니다 (길면 1~2분정도 걸림)
7. 출력된 내용을 보면 IP Address라는 값이 있는데, 이걸 준비물로 준비해둔 컴퓨터의 인터넷 브라우저의 주소창에 입력합니다. (이 때도 꼭 같은 공유기로 인터넷에 연결된 컴퓨터여야합니다.)
- 이 때 만약 "Your connection is not private" 경고창이 뜬다면, 왼쪽 아래의 Advanced를 누르고 Proceed to 192.168.0.X (unsafe) 와 버튼을 누릅니다.
8. Firmware Update, Cloud Services, Advanced Settings 같은 창이 뜨면 Advanced Settings를 눌러주세요.
9. 오른쪽 위에 Administrator login을 누릅니다.
10. 비밀번호를 치라고 나오는데, 영수증 출력기 옆에 붙어있는 스티커의 SERIAL No.을 보고 입력하면 됩니다.
11. 로그인에 성공하면 Product Status 창이 뜹니다.
12. 왼쪽 TM-Intelligent 아래에 EpsonNet Config for TM-i 버튼을 누릅니다.
13. 새로 뜬 창에서 Server Direct Print를 클릭합니다.
14. Server Direct Print를 enable로 바꾸고, 아래에 값들을 세팅합니다. 이 때, **siteId**라는 곳에는 현재 설치중인 지점이 몇호점인지를 생각해보고 그 값에 1을 더하면 됩니다. (예를 들어, 강남점: 2, 잠실점: 3, 강동점: 4, …, 금천점: 8 등등)
- Server1 에 URL: https://api.blitz-dynamics.com/bill-printer/site/**siteId**
- 예를 들어, 금천점은 https://api.blitz-dynamics.com/bill-printer/site/8
- Server1 에 Interval: 1
- ID: ks**앞에 0을 붙여서 4자리로 만든 siteId**printer
- 예를 들어, 강남점: ks**0002**printer, 강동점: ks**0004**printer, 금천점: ks**0008**printer, 10호점이 생기면: ks**0011**printer
- 나머지는 그대로 둡니다.
15. 입력이 끝나면, 입력한 페이지를 스크린샷을 찍어서 개발팀에게 전달합니다.
16. 오른쪽 위에 Apply & Restart 버튼을 클릭하고 확인을 눌러 설정 변경을 적용합니다.
- 적용중 창이 떠서 시간이 흐르다가 완료입니다.
