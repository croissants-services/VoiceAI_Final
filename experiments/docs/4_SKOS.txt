# SKOS에서 품절을 걸고 싶어요
영업중 품절을 거는 방법에 대해서 설명 해드리겠습니다. 첫째, 주문관리 페이지의 하단에서 자재 품절을 클릭합니다. 둘째, 품절과 관계된 제품을 드래그로 찾거나 검색으로 찾아서 다음을 클릭합니다. 셋째, 특정 제품을 선택합니다. 이때 무엇 때문에 품절이 발생하였는지 확인이 가능합니다. 예를들어, 깐쇼새우_31/40이 품절이라 품절을 걸게되면 깐쇼새우 관련 제품 모두가 품절이 걸리게 됩니다. 
특정 물품을 품절 처리하는 것도 가능합니다. 특정 제품만 품절 처리할 경우, 품절 설정 화면에서 해당 제품을 선택하여 품절 처리할 수 있습니다. 
일부 품목만 품절 처리하려는 경우 전체 품목에 대해 품절이 적용되지 않는다면, 우선 품절 해제 후 다시 품절 처리해 주세요. 품절 처리가 완료되면 반드시 화면에서 확인해야 합니다. 품절 완료될 때까지 기다려 주세요. 
품절 처리 오류 발생 시 메세지가 나오면서 품절이 처리가 되지 않을 수 있습니다. 이럴경우에 메세지를 공유하여 본사에 요청 부탁드립니다. 최대한 빠르게 해결하겠습니다. 

# 배달대행사가 운행을 멈추었습니다.
천재지변 등으로 배달 대행사가 운행이 힘든 경우 또는 단축 운행하는 경우.
각 지점마다 배달대행 앱이 설치되어 있습니다. 배달 공지 사항이 발생할 경우, 해당 공지 메세지 화면을 캡처하여 본사 요청 방에 올려주시면 빠르게 처리 도와드리겠습니다. 예를 들어, “석촌지점장입니다. 금일 단축 운영 안내 드립니다. 설천과 도로결빙으로 인해 배송이 어려워 조기마감 예정입니다. 오후 16시59분까지 배차되는 콜까지 수행 끝으로 조기마감되오니 참고 부탁드립니다. 운영재개일 29일 명절당일휴무 30일 오전 7시 정상운영입니다”와 같은 [송파석촌지점 공지]조기 마감 안내를 송파_04_본사에_요청 방에 올려주시면 되겠습니다. 

# 배달 대행사가 시간을 늘려 달라고 요청하였습니다.
배달 대행사가 인원부족등으로 시간을 늘려달라고 요청한 경우, 왼쪽에 있는 생산 란에서 주방 운영 모니터 란으로 이동합니다. 배달 예상 시간을 배달 대행사 및 슈퍼바이저와 함께 소통하여 변경합니다. 

# 출근 메시지가 오지 않습니다.
현장에 출근하였는데 출근메세지가 오류등으로 인하여 메세지가 오지 않은 경우, https://skos.blitz-dynamics.com/userHome/home 로 접속 후 로그인하면 공지사항 및 출근의 여부를 확인할 수 있는 메세지를 볼 수 있습니다. 

# 출근 버튼을 늦게 눌렀습니다.
출근 버튼을 늦게 누른 경우 출근 버튼은 출근 2시간전부터 활성화 되오니 가급적 현장에 도착하자마자 눌러주시길 바랍니다. 다만 출근 버튼을 늦게 누른 경우 스케줄 방에 리더님이 “천재환 멤버님 정시 출근하였으나 늦게 출근 눌렀습니다” 등으로 남겨주실 수 있도록 전달해주시면 감사하겠습니다. 

# 스케줄이 변경되었는데 반영이 안되었어요.
스케줄이 변경되었는데 스케줄표에 반영이 되지 않은 경우 현장의 리더님께 요청 해주세요. 

# SKOS 캐시, 데이터 지우기
샤오미태블릿에서 설정창에 접속 후 앱을 클릭한 후 앱 관리를 클릭하여 앱관리 페이지에 접속합니다. 다음으로 생산관리 시스템을 클릭합니다. 모든 데이터 지우기 및 캐시 지우기를 클릭하면 됩니다.
